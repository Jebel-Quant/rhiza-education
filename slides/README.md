# Slides

## `rhiza-intro.md` — 30-minute introduction to Rhiza

A Marp presentation covering the repo zoo problem, how Rhiza works, getting started, and the ecosystem.

### Build to PDF or HTML

```bash
# HTML (self-contained)
uvx --from marp-cli marp rhiza-intro.md --html

# PDF
uvx --from marp-cli marp rhiza-intro.md --pdf

# Watch mode (live preview while editing)
uvx --from marp-cli marp --watch rhiza-intro.md
```

Or use the [Marp for VS Code](https://marketplace.visualstudio.com/items?itemName=marp-team.marp-vscode) extension — open the file and click **Open Preview**.
